◆ bspI2CExist()

bool bspI2CExist ( uint8_t address)

Checks if an I2C device is present on the bus.

This function verifies the presence of an I2C device by initiating a transmission and checking for an acknowledgment.

Parameters
[in]addressDevice's I2C address.
Attention
Ensure that the I2C bus is initialized before calling this function.
Returns
  • true: Device is present.
  • false: Device is not responding.

Detailed Description

Ultrasonic sensor class for measuring distances.

This class provides functionality to interact with an ultrasonic sensor, measure the time duration of the echo signal, and calculate distances in different units.

Initializes the Ultrasonic sensor with the specified trigger and echo pins, and sets the timeout value.

This constructor sets up the ultrasonic sensor by assigning the trigger and echo pins and configuring them as output and input respectively. It also sets the timeout for the pulse measurement.

Parameters
[in]_triggerPinThe pin number connected to the trigger pin of the ultrasonic sensor.
[in]_echoPinThe pin number connected to the echo pin of the ultrasonic sensor.
[in]timeOutThe timeout value for the pulse measurement in microseconds.
[out]None
[in,out]None
Attention
Ensure that the pin numbers passed are correctly connected to the ultrasonic sensor. The timeout should be set according to the maximum expected distance measurement, default is 23000 microseconds.

Detailed Description

Handles button presses, managing debouncing, click detection, and hold events.

This class provides functionality for:

  • Debouncing button inputs to avoid unintended multiple presses.
  • Detecting single-click, double-click, and hold events.
  • Managing callback functions for different button interactions.
  • Utilizing a finite state machine (FSM) for event handling.

The class allows users to configure various parameters such as debounce duration, double-click intervals, and hold duration. It continuously updates button states and triggers appropriate event callbacks as defined.

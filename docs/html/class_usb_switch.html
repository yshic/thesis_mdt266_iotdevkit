Detailed Description

USB Switch control class for managing devices operations.

This class provides methods to control a usb switch module, including setting the output value for each output, the usb switch module has two output available: USB_OUTPUT1 and USB_OUTPUT2 toggling the output state, checking the status of the output, and setting the output value as a percentage.

Attention
  • If using the grove connector, the yellow wire will control USB_OUTPUT1 and the white wire will control USB_OUTPUT2.

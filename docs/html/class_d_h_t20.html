dht20_error_t DHT20::begin ( )

Initializes the DHT20 sensor.

This function initializes the DHT20 sensor by waiting for a brief period after power-on and then resetting the sensor.

Parameters
[in]None
Attention
Ensure that the sensor is correctly connected and powered on before calling this function. This function includes a 100ms delay to allow the sensor to stabilize after power-on.
Returns
  • DHT20_OK: Initialization success.

dht20_error_t DHT20::readTempAndHumidity ( )

Reads temperature and humidity data from the DHT20 sensor.

This function attempts to read temperature and humidity data from the DHT20 sensor. It retries the operation multiple times if the sensor is not ready or data is not available. The raw sensor values are then converted to floating-point temperature and humidity values.

Parameters
[in]None
Attention
Ensure that the sensor is correctly connected and that I2C communication is functioning. This function may block for short periods while waiting for the sensor to be ready and for data to be available.
Returns
  • DHT20_OK: Success
  • DHT20_TIMEOUT: Timeout, the sensor is not ready or data is not available after multiple attempts.

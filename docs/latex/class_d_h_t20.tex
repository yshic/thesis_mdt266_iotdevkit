Reads temperature and humidity data from the \doxylink{class_d_h_t20}{DHT20} sensor. 

This function attempts to read temperature and humidity data from the \doxylink{class_d_h_t20}{DHT20} sensor. It retries the operation multiple times if the sensor is not ready or data is not available. The raw sensor values are then converted to floating-\/point temperature and humidity values.

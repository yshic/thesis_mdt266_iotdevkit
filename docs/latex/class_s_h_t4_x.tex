\doxysubsection{Detailed Description}
Class for interfacing with the \doxylink{class_s_h_t4_x}{SHT4X} temperature and humidity sensor. 

The \doxylink{class_s_h_t4_x}{SHT4X} class provides methods to initialize, configure, and retrieve temperature and humidity data from the \doxylink{class_s_h_t4_x}{SHT4X} sensor over an I2C interface.

sht4x\+\_\+error\+\_\+t SHT4\+X\+::update (\begin{DoxyParamCaption}\item[{void}]{}{}\end{DoxyParamCaption})}



Updates sensor readings from the \doxylink{class_s_h_t4_x}{SHT4X}. 

This function sends a measurement command to the \doxylink{class_s_h_t4_x}{SHT4X} sensor, waits for the required conversion time, reads the data, verifies the checksum, and updates temperature and humidity values.

\begin{DoxyAttention}{Attention}
Ensure that the sensor is correctly connected to the I2C bus and that appropriate delays are provided for accurate readings.
\end{DoxyAttention}

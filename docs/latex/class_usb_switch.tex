\doxysubsection{Detailed Description}
USB Switch control class for managing devices operations. 

This class provides methods to control a usb switch module, including setting the output value for each output, the usb switch module has two output available\+: USB\+\_\+\+OUTPUT1 and USB\+\_\+\+OUTPUT2 toggling the output state, checking the status of the output, and setting the output value as a percentage.

\begin{DoxyAttention}{Attention}

\begin{DoxyItemize}
\item If using the grove connector, the yellow wire will control USB\+\_\+\+OUTPUT1 and the white wire will control USB\+\_\+\+OUTPUT2. 
\end{DoxyItemize}
\end{DoxyAttention}
